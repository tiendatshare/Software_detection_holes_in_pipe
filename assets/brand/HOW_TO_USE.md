# Hướng dẫn thay logo / Logo Replacement Guide / 로고 교체 가이드

## Tiếng Việt
1. Đặt file ảnh logo vào thư mục này (`assets/brand/`)
2. Đặt tên file là: `logo.png`, `logo.jpg`, hoặc `logo.svg`
3. Kích thước khuyến nghị: **200×60px**, nền trong suốt (PNG)
4. Khởi động lại app → logo tự động hiển thị trên toolbar

## English
1. Place your logo image in this folder (`assets/brand/`)
2. Name the file: `logo.png`, `logo.jpg`, or `logo.svg`
3. Recommended size: **200×60px**, transparent background (PNG)
4. Restart the app → logo automatically appears in the toolbar

## 한국어
1. 로고 이미지를 이 폴더에 넣으세요 (`assets/brand/`)
2. 파일 이름: `logo.png`, `logo.jpg`, 또는 `logo.svg`
3. 권장 크기: **200×60px**, 투명 배경 (PNG)
4. 앱 재시작 → 툴바에 로고가 자동으로 표시됩니다
